#MCU_Studio


Ubuntu Linux AVR 集成开发环境IDE，类似于 Atmel 官方的 Windows 版本集成开发坏境 Atmel Studio 6.0（ MCU Studio AVR Studio）

主要功能：

1、编写C程序源码；

2、编写C程序头文件；

3、将程序文件编译成链接O文件；

4、将链接文件编译成Hex文件（Intel HEX）；

5、将Hex文件下载到目标单片机；

6、集成一些单片机常用工具，例如：COM口通讯、文件传输

7、逐步优化中。。。

运行坏境：

1、Linux Ubuntu
